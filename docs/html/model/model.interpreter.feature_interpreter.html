model.interpreter.feature_interpreter module¶

class FeatureInterpreter(feature_names: List[str])[source]¶

Bases: object

Class Feature Interpreter.

The class is used to generate the following information:

  • feature distribution

  • feature correlation

  • feature importance ranking for a trained model
