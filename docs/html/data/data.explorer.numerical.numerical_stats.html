data.explorer.numerical.numerical_stats module¶

class NumericalStats(min: Union[float, int, None] = None, max: Union[float, int, None] = None, mean: Optional[float] = None, median: Union[float, int, None] = None, sd: Union[float, int, None] = None, histogram: Optional[List[Tuple[Union[float, int, None], Union[float, int, None], int]]] = [], kde: Optional[List[Tuple[Union[float, int, None], Union[float, int, None]]]] = [], total_count: Union[float, int, None] = 0, nan_count: Union[float, int, None] = 0)[source]¶

Bases: xai.data.abstract_stats.AbstractStats

NumericalStats contains following basic information:

  • _total_count: total count of values

  • _nan_count: total count of nan values

  • _min: minimum of all values

  • _max: maximum of all values

  • _mean: mean of the values

  • _median: median of the values

  • _sd: standard deviation of the values

  • _histogram: a histogram of value distribution represented by a list of (x_left, x_right, count)

  • _kde: a kernel density estimation curve represented by a list of points
