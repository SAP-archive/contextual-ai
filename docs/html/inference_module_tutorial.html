Contextual AI Explainer tutorials¶

This directory houses the Jupyter notebooks which demonstrate how to use each explanation algorithm found in xai.explainer.

  • lime_tabular_explainer.ipynb:

    • Explanation algorithm: LIME tabular explainer

    • Target model: RandomForest

    • Dataset: Wisconsin breast cancer

  • lime_text_explainer.ipynb:

    • Explanation algorithm: LIME text explainer

    • Target model: Naive Bayes

    • Dataset: 20newsgroups

  • lime_text_explainer_with_keras.ipynb:

    • Explanation algorithm: LIME text explainer

    • Target model: Convolutional Neural Network

    • Dataset: 20newsgroups

  • shap_tabular_explainer.ipynb:

    • Explanation algorithm: SHAP kernel explainer

    • Target model: RandomForest

    • Dataset: Wisconsin breast cancer
